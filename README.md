

Install

`yarn add react-tailwind-datetime-picker`

In tailwind config:
```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './src/**/*.{js,jsx,ts,tsx}',
    './node_modules/react-tailwind-datetime-picker/dist/index.esm.mjs',
  ],
  theme: {
    extend: {},
  plugins: [],
}```
